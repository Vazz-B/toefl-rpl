# Sistem Manajemen Pendaftaran TOEFL Berbasis Web
## UPT Bahasa Universitas Trunojoyo Madura

Sistem pendaftaran TOEFL online yang dibangun menggunakan PHP, MySQL, Bootstrap 5, dan Apache.

### Fitur Utama
- **Mahasiswa**: Registrasi, login, pilih jadwal, upload bukti bayar, download kartu peserta, lihat hasil skor
- **Admin**: Kelola jadwal, verifikasi pembayaran, input skor, kelola data peserta

### Cara Menjalankan
1. Install XAMPP (Apache + MySQL + PHP)
2. Copy folder `toefl-registration` ke `htdocs`
3. Import `database.sql` ke MySQL
4. Akses `http://localhost/toefl-registration`
